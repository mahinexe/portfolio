# Portfolio Website

![Portfolio Preview](./img/thumbnail.png)

A modern, coder-themed portfolio website featuring TypeScript-driven animations and terminal aesthetics.

## 🚀 Features

- **Terminal-Themed Design**: Console-style code blocks with macOS window controls
- **TypeScript Animations**: Dynamic typewriter effects and smooth scroll reveals
- **Fully Responsive**: Optimized for all devices (mobile, tablet, desktop)
- **Clean Architecture**: Custom CSS without framework dependencies
- **Performance Optimized**: Fast loading with minimal dependencies

## 🛠️ Tech Stack

- HTML5
- Custom CSS3
- TypeScript
- ES2020 Modules

## 📦 Project Structure

```
├── css/
│   └── style.css          # All custom styles
├── dist/
│   └── main.js            # Compiled TypeScript
├── img/                   # Images and assets
├── src/
│   └── main.ts            # TypeScript source
├── index.html             # Main HTML file
├── package.json           # Dependencies
└── tsconfig.json          # TypeScript configuration
```

## 🚀 Getting Started

### Prerequisites

- Node.js and npm installed

### Installation

```bash
# Clone the repository
git clone https://github.com/mahinexe/portfolio.git

# Navigate to project directory
cd portfolio

# Install dependencies
npm install

# Build TypeScript
npm run build
```

### Development

```bash
# Watch mode for TypeScript
npm run build:watch
```

## 📱 Responsive Design

The portfolio includes 5 responsive breakpoints:
- **1400px+**: Large desktops
- **980px**: Tablets/small laptops
- **760px**: Mobile landscape
- **540px**: Standard phones
- **380px**: Small phones

## ✨ Features Highlight

- **Animated Hero Section**: Dynamic role typing with "Tech Enthusiast/Nerd/Geek"
- **Terminal Code Blocks**: Syntax-highlighted TypeScript snippets
- **Scroll Reveal Animations**: Smooth fade-in effects on scroll
- **Contact Typewriter**: Animated contact message
- **Project Showcase**: Grid layout with hover effects
- **Education Timeline**: Professional education cards
